# 公安指挥任务控制台 Wireframe V2

## Overview

V2 当前方案采用“左侧卡片工作区 + 右侧智能协同侧栏”的布局。页面用于公安指挥大厅或警员工作台，视觉定位是高密度指挥工具：稳定、可扫读、少解释文案，不做营销页或炫技大屏。

核心目标：

- 左侧是业务卡片工作区，只展示任务执行后产生的业务结果。
- 右侧是智能协同侧栏，上半区为 AI 对话，下半区为流程列表。
- 右侧侧栏可收起为悬浮“智能协同”入口，入口在右侧，可上下拖动，点击展开。
- 页面固定为浏览器视口高度，不让 body 或底部无限延伸。
- 所有滚动区域都可滚动，但不显示滚动条。

## Components

- 顶部标题栏：只展示产品名称“公安指挥任务控制台”。
- 卡片工作区：视频、地图、表格、文本、JSON 等业务卡片。
- 卡片操作：每张卡片右上角只保留图形按钮，`▴ / ▾` 控制收起展开，`□ / ×` 控制最大化恢复。
- 智能协同：AI chat 消息区和底部 composer。
- Composer：输入框单行起步，右侧图形按钮独立固定高度；输入多行时不撑高按钮。
- 预案指引：流程阶段和子动作列表。
- 流程状态：状态只通过子项前置圆形符号体现，不在每行末尾显示“已完成/执行中/等待”等文字。

## Collapsed Side Panel

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│ [警] 公安指挥任务控制台                                                       │
├──────────────────────────────────────────────────────────────────────────────┤
│ 卡片工作区                                                        智能协同    │
│                                                                  悬浮入口    │
│                                                                  可上下拖动  │
└──────────────────────────────────────────────────────────────────────────────┘
```

Collapsed behavior:

- 右侧侧栏宽度变为 `0px`，不占用卡片区域宽度。
- 悬浮入口只显示竖排“智能协同”文字。
- 悬浮入口可上下拖动，点击文字展开右侧侧栏。
- 拖动时不触发展开。

## Card Behavior

- 卡片在对应任务有结果后出现，不提前占位等待状态。
- 卡片右上角不展示任务状态 badge。
- 卡片默认高度统一为 `360px`。
- 卡片工作区使用 2 列网格；宽屏可扩展为 3 列，窄屏降为 1 列。
- 卡片内部内容可滚动或横向适配，但滚动条不显示。
- 收起卡片只保留标题栏。
- 最大化卡片覆盖卡片工作区，不覆盖右侧智能协同侧栏。

## Smart Collaboration Panel

右侧侧栏由上下两个等高区域组成：

- `智能协同`：上半区，显示 AI 对话。
- `预案指引`：下半区，显示流程阶段和动作。

视觉规则：

- 侧栏整体背景与卡片区不同。
- 两个区域之间保留 `8px` 间隔。
- 两个区域各自有边框、圆角和略深底色，明确表现为同一侧栏里的上下两块。
- 两个区域都独立滚动，滚动条不显示。

## Composer

- Composer 固定在智能协同区域底部。
- 输入框单行起步，高度为 `38px`。
- 输入框可以增长到多行，但右侧按钮保持 `38px × 38px`，不被 textarea 撑高。
- 运行中按钮显示停止图形 `■`。
- 未运行时后续可切换为发送图形按钮。

## Flow Status

流程子项不使用文字状态列。状态通过每个子项前面的圆形符号表达：

- 已完成：圆圈内 `√`。
- 执行中：loading 旋转圆圈。
- 失败：圆圈内 `×`。
- 等待：空圆。
- 已停止：圆圈内 `!`。

选中的流程子项使用行背景和左侧色条提示。

## Scrolling

- 页面主体固定在 `100dvh`。
- 卡片工作区独立滚动。
- AI 消息区独立滚动。
- 预案指引独立滚动。
- 表格、JSON 等卡片内部可滚动。
- 所有滚动区域隐藏滚动条，但保留滚轮、触控板和触摸滚动能力。

## Accessibility Notes

- 悬浮入口、收起、最大化按钮保留 `aria-label`。
- 系统回执区域使用 `aria-live="polite"`。
- 按钮保留可见 focus ring。
- 状态不能只依赖颜色，流程状态符号需保留语义映射。
